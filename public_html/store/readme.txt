Cash on delivery Fees.

For apply this extension, Please do the following steps.

1. Extract zip file.
2. Upload the admin & catalog folders.

After finish above steps extension installed into website succussfully, now for making it's live do following steps.

3. Go to Extensions >> Order Totals.
4. Install Cash on delivery Fees Module.
5. After installing, click on the edit button (icon pencil), & make status enable, also set fee type & fees amount, if you not set any amount then it set 0.

6. After set products, save it.

After that whenever any customer choose cash on delivery as payment option, then fees apply respectivaly.

Enjoy it.
